# Product Carousel Challenge

A responsive product carousel with interactions and custom scrollbar styling.

Preview link: https://raw.githack.com/JiayueChen/carousel-challenge/main/index.html

## Tech Stack
- HTML (Semantic)
- Vanilla JavaScript
- TailwindCSS v4.1.18 via Tailwind CLI

## Features
- 10 product cards with ratings and pricing
- Includes 3 font families with responsive typography
- Responsive design:
    - Mobile: 2-column grid, first 4 products visible, "Show More" button with slide-down animation
    - Desktop: Horizontal scrollable carousel and "Shop All Best Sellers" button
- Custom scrollbar: 4px thumb height (default), 6px on hover
- Accessibility: ARIA labels, semantic HTML, screen reader support
